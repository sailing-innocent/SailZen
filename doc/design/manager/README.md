# Daily Manager

> SailZen 日常管理系统 - 模块设计文档目录

日常管理器使用一套前后端全栈开发框架来维护日常生活，涵盖财务、健康、项目任务、物资和文本内容等多个管理模块。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Frontend (packages/site)                  │
│  React + TypeScript + Vite + Zustand + Tailwind CSS         │
├─────────────────────────────────────────────────────────────┤
│  Pages: main | money | health | project | content | text    │
│  Components: 业务组件 + UI 组件 (Radix UI)                   │
│  Store: 各模块独立 Zustand Store                             │
│  API Client: lib/api/*.ts                                    │
├─────────────────────────────────────────────────────────────┤
│                    Backend (sail_server)                     │
│  Python + Litestar + SQLAlchemy + PostgreSQL                 │
├─────────────────────────────────────────────────────────────┤
│  Controllers: 请求处理和参数验证                              │
│  Models: 业务逻辑实现                                         │
│  Data: ORM 模型定义                                          │
│  Utils: 工具函数                                             │
└─────────────────────────────────────────────────────────────┘
```

## 模块详细说明

### 1. 财务管理 (life_budget.md) ✅

- **账户管理**：多账户支持，余额追踪，余额校准
- **交易记录**：收支记录，标签分类，时间筛选，分页查询
- **预算管理**：预算规划，核销跟踪，使用率分析
- **统计分析**：收支统计，趋势图表，多维度分析

### 2. 项目管理 (project.md) ✅

- **项目管理**：项目 CRUD，状态机，时间规划（季度双周）
- **任务管理**：任务 CRUD，状态转换，截止日期，延期功能
- **提醒功能**：即将到期任务，逾期任务，主页 TODO List
- **待实现**：里程碑、日程管理、进度追踪

### 3. 健康管理 (health.md) 🔶

- **已实现**：体重记录，趋势图表
- **设计中**：健身管理，口腔/皮肤保养，医疗保险

### 4. 物资管理 (necessity.md) 📋

- **设计中**：住所管理，物资管理，库存跟踪，旅程管理，过期提醒

### 5. 文本管理 (text.md) ✅

- **作品管理**：作品 CRUD，元数据管理
- **版本管理**：多版本支持，章节列表
- **内容管理**：章节阅读，内容搜索，文本导入
