# SailZen 产品需求文档

> 版本：v1.0 | 更新：2026-02

## 1. 产品概述

SailZen 是面向个人用户的全方位生活管理系统，整合知识管理、财务追踪、健康监控、项目规划等功能。

### 核心组件

| 组件 | 技术栈 | 说明 |
|------|--------|------|
| sail_server | Python + Litestar + PostgreSQL | 后端 API 服务 |
| site | React + TypeScript + Vite | Web 前端 |
| vscode_plugin | TypeScript + VSCode API | 笔记管理插件 |

---

## 2. 功能模块

### 2.1 财务管理 ✅

个人财务全面管理。

- 多账户管理与余额追踪
- 收支交易记录与分类（标签系统）
- 预算规划与执行监控
- 财务统计与可视化分析

### 2.2 项目管理 ✅

项目与任务的全生命周期管理。

- 项目/任务 CRUD
- 状态机管理（项目7状态，任务5状态）
- 任务延期与截止日期追踪
- 即将到期/逾期任务提醒

### 2.3 健康管理 🔶

个人健康数据追踪。

- ✅ 体重记录与趋势分析
- ✅ 目标体重预测
- 📋 健身管理（待实现）
- 📋 健康用品库存（待实现）

### 2.4 文本内容管理 ✅

长篇文本的存储、索引与分析。

- 作品、版本、章节管理
- 文本导入与内容搜索
- LLM 辅助分析（大纲、人物、设定提取）
- 任务调度与进度追踪

### 2.5 物资管理 ✅

多住所物资定位与库存追踪。

- 住所与容器管理
- 物资分类与库存追踪
- 旅程携带物资管理
- 低库存预警

### 2.6 笔记知识管理 ✅

基于 Dendron 的层级化笔记系统。

- 层级笔记导航（点号分隔）
- Wiki 风格链接
- 多 Vault 工作区
- Zotero 文献集成
