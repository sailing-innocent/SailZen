# 章节类型参考

## 标准章节 (standard)

常规编号章节，最常见类型。

### 匹配模式
- 第[数字/中文数字]章
- 第[数字/中文数字]节
- Chapter [数字]
- CHAPTER [数字]
- [数字]. [标题]

### 示例
```
第一章 风起云涌
第1章 初入江湖
Chapter 1 The Beginning
1. 新的开始
```

## 前置章节 (prologue)

正文开始前的引导内容。

### 常见标题
- 楔子
- 序章 / 序言 / 序
- 前言 / 引言
- Prologue
- Preface
- Introduction
- 开篇
- 引子

### 处理规则
- 排序在标准章节之前
- 编号可设为 0 或保持原样
- 通常较短，字数可能明显少于标准章节

## 过渡章节 (interlude)

章节之间的插曲内容。

### 常见标题
- 间章
- 插曲
- Interlude
- 幕间
- 过渡

### 处理规则
- 按出现位置插入标准章节之间
- 保留原有顺序

## 后置章节 (epilogue)

正文结束后的补充内容。

### 常见标题
- 尾声
- 后记
- 结局
- Epilogue
- Afterword
- 完结篇 / 大结局
- 终章

### 处理规则
- 排序在标准章节之后
- 编号可设为最大值或保持原样

## 番外章节 (extra)

主线故事之外的衍生内容。

### 常见标题
- 番外
- 外传
- 外传篇
- Extra
- Side Story
- 特别篇
- 特典
- 附录

### 处理规则
- 通常排在最后
- 可能包含多个独立小节
- 可统一归类为番外系列

## 作者相关内容 (author)

作者与读者的互动内容。

### 常见标题
- 作者的话
- 作者感言
- 写在前面
- 写在后面
- 上架感言
- 完本感言
- 请假条
- 更新说明

### 处理规则
- **通常建议过滤掉**，不属于正文内容
- 如保留，单独归类为"作者相关"
- 字数通常很短（<500字）

## 非章节内容 (noise)

需要过滤的噪音内容。

### 常见类型
- 广告推广
- 平台信息
- 版权声明（重复出现）
- 网址链接
- 扫码提示
- 重复章节标题
- 空行堆积

### 识别特征
- 包含 URL / 网址
- 包含"关注"、"扫码"、"加群"等营销词汇
- 纯符号组成
- 极短内容（<100字）且无意义
- 与其他章节内容重复

## 章节标题变体参考

### 数字表示法
```
第1章 / 第一章 / 第壹章 / 第①章
1. / 1、/ 一、/ 壹、
Chapter 1 / CHAPTER ONE / Ep.1
第1话 / 第1回 / 第1集
```

### 特殊格式
```
【第一章】风云
卷一·第一章
第一卷 第一章
第1章：标题
第1章——标题
```

### 无编号标题
```
风云初起（需根据上下文判断是否章节）
新的开始
...
```

## 排序规则

当需要为章节分配数字序号时，建议顺序：

```
0: 楔子/序章
1-N: 标准章节（第1章到第N章）
N+1: 尾声/后记
N+2~: 番外/外传
```

对于非数字标题的排序，可以：
1. 按在原文中出现的先后顺序
2. 根据类型归类排序（前置→标准→后置→番外）
3. 保留原始标题文本作为显示名
