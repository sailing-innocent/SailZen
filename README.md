# SailZen 

**Just Sail Then!** SailZen是一个基于大语言模型的世界观引擎，旨在帮助带有庞大世界观的长篇小说，复杂剧本，大型游戏等虚构/半虚构创作提供一个基于LLM的工具，支持
- 从现有的长篇文本中提取世界观概念（人物、地理、设定、组织、情节、理念等……）并支持丰富的检索和可视化功能
- 辅助从零开始的世界观设定，让创作者可以与LLM合作，让LLM在后台辅助拓展，按需审阅合并，灵活修改，审查设定矛盾，丰富设定细节，提出剧情建议等

## Motivation 动机

小说，动画，游戏等娱乐产业常常附带很多新奇有趣的世界观设定，但是个人创作者精力难以兼顾，团队合作又难以对他人工作熟悉配合，因此绝大部分的作品的世界观都有不够严谨，难以拓展的缺点。就算是创作者自己来续写都很难照顾全部伏笔和设定构思，同人OOC的现象更是司空见惯。而传统依靠读者，编辑，作者的人力筛选框架很难维护一个完整清晰的世界观，依靠作者个人的知识积累也很难做到彼此照应，细节严谨，更没有足够的精力和动力来判断每一段新创作的文本是否符合原来的世界观设定，或者找到原本世界观设定中彼此冲突和不合理的部分。2024年推理模型的火热让
